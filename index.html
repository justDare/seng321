Mission

We at QuantumGPS are committed to providing the most scalable, functional and user friendly ocean network mapping on the market. Through the use of floating sonobuoys and a node network mapping across the ocean floor, we can use acoustic experiments to map, track and locate objects such as water vessels, marine mammals and torpedos.

Current Project

Project DND aims to improve the existing inhouse software which controls data collection and tracking of water buoys in a specific region. The company currently has a stable version but other features are required to improve the functionality and scalability of the system.
